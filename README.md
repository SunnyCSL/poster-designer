# 🎨 Poster Designer

A Canva-like multi-layer poster design tool built with Fabric.js.

**Cross-platform** | **PWA** | **Dark Theme** | **Google Slides export**

## Features

- Multi-layer canvas (images, text, shapes, lines)
- Template sizes (A3, A4, Square, Portrait, Mobile, Wide)
- Drag, resize, rotate, reorder any element
- Undo/Redo (⌘Z / ⌘⇧Z)
- Export as PNG (2x)
- Google Slides push (coming soon)
- Auto-save to localStorage
- PWA — installable on iPhone/Android/Desktop

## Usage

Open `index.html` in any browser. No server required.

## Development

```bash
git clone https://github.com/SunnyCSL/poster-designer
cd poster-designer
# Edit index.html directly — no build step needed
```
