# Changelog

## [v0.1.0] — 2026-05-16

### Added
- **Canvas Engine**: Fabric.js v5.3.1 — multilayer canvas with drag/resize/rotate
- **Template Sizes**: A3, A4, Square 1:1, Portrait 4:5, Mobile 9:16, Wide 16:9, Instagram Story
- **Image Upload**: Local file picker with auto-scale to canvas
- **Text Tool**: Editable textbox with font size, color, bold, italic
- **Shape Tool**: Rounded rectangle (draggable, resizable, color picker)
- **Line Tool**: Horizontal line element
- **Layer Panel**: Full layer list with visibility toggle, delete, reorder (click to select)
- **Properties Panel**: Position (X/Y), Size (W/H), Rotation, Opacity, Text-specific controls
- **Undo/Redo**: History stack (50 steps), ⌘Z / ⌘⇧Z keyboard shortcuts
- **Zoom**: Zoom in/out/fit, keyboard accessible
- **Export PNG**: 2x resolution download
- **State Persistence**: localStorage auto-save every 5s, survive refresh
- **PWA**: Manifest + Service Worker, offline-capable, installable to home screen
- **Dark Theme**: Eye-friendly dark UI, responsive layout (desktop + mobile)
- **Delete**: Delete/Backspace key, right-click delete button
- **Multi-select**: Fabric.js native multi-selection with shift-click
