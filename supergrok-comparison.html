🤖 我哋系統使用可行性

Lite $10 — ❌ 唔建議做 Main Model

Rate limits 太 tight — 我一日十幾個 sessions，每個 session 幾十個 tool calls，好快 hit 429。
Grok 4 basic 唔係 4.3，用唔到 DeepSearch / Big Brain / Voice。480p image 對我哋 poster work 冇用。
最佳用法：只係試水，或者用嚟做 X Search 同 TTS（呢兩個功能獨立運作）

Standard $30 — ✅ 可以做 Main Model（但要考慮成本）

Grok 4.3 + DeepSearch + high rate limits — 足以取代 DeepSeek 做 system default model。
但係 $30/month + 我哋而家 DeepSeek/MiniMax 成本 ($0-5) = 貴咗 $25-30/月
最好用法：用 Standard 嘅 DeepSearch + X Search 做輔助，唔 replace 而家嘅 models

💰 Cost Scenarios

Scenario A — 而家 Keep 不變

DeepSeek V4 Flash（main）+ MiniMax M2.7（auxiliary）+ Kimi K2.6（coding）
成本：~$0-5/month
好處：已經運作順暢，所有功能齊全
缺點：冇 xAI 嘅 DeepSearch

Scenario B — Lite + 而家嘅 setup

$10/month 加 xAI access，用嚟做 X Search + TTS
Main model 繼續 DeepSeek / MiniMax
成本：~$10-15/month
推薦：試一個月，睇下 xAI 生態值唔值得 upgrade

Scenario C — Standard + 而家嘅 setup

$30/month 加 Grok 4.3 + DeepSearch
Main model 可能轉 Grok 4.3，DeepSeek 做 fallback
成本：~$30-35/month
推薦：如果你發現 DeepSearch 真係好有用
